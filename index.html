Newflix

Newflix is a Netflix Clone, a full stack project that I worked on to sharpen my skills in front and backend development. The function clone allows for users to watch and stream movie trailers once registered and signed in. The backend database is developed on PostgreSQL, the backend is developed through Ruby on Rails, and the frontend is developed using React/Redux. Cloud storage for images and videos is supported through Amazon AWS S3.

Monster Mayhem

Pokemon-esque battle arena built using React, Express, and MongoDB. Features villains from popular horror movie franchises. Players may build a two-man team then face off against each other or against the AI using their selected characters. As frontend lead, I implemented Axios and Javascript promises to pull information from MongoDB, using Redux to pass information into React components for improved UI/UX and game logic. Integrating websockets allows for temporaries servers to create online games between two players.

Interdimensional Sniper

Interdimensional Sniper is a side-view sniper game, where the player's objective is to survive as long as possible by sniping enemy creatures crossing horizontally on the screen. The user must use the mouse to aim and left click on the mouse to shoot a laser at the marching enemies before they cross over from the left to right side of the game screen.
